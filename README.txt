Para o pleno funcionamento do programa, tu deve ter o Windows
server instalado no teu pc. Caso tu não tenha o windows server
no pc instale.

Passo 1: clique em ADD NEW QUERY

Passo 2: va na nossa pasta banco de dados, depois estrutura.
agora copie e cole todo o texto na new query e execute

Passo 3: na pasta banco de dados abra o conexão.udl e localize
teu banco de dados utilizando o nome do seu servidor win server.

Passo 4: após encontrado teu banco de dados, de um ok. Após isso
abra com o bloco de notas e pegue a connection string.

Passo 5: Já com a connection string entre o no codigo fonte 
abra a classe conexao.cs e mude a connection string.

obs.:Se ocorrer um erro na connection string em uma barra adicione
outra igual.
